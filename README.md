# Food Delivery Application (Java Swing)

This project is a desktop-based Food Delivery Application developed using Java Swing. It simulates a basic food ordering system with multiple pages, user authentication, and file-based data storage.

## Features

- User Registration and Login system  
- File-based storage using text files  
- Multi-page GUI navigation  
- Product categories (Food, Drinks, Fruits, etc.)  
- Item selection and basic cart functionality  
- Account management (view and delete profile)  
- Help/Issue reporting system  

## Technologies Used

- Java  
- Java Swing (GUI)  
- File Handling (BufferedReader, BufferedWriter)  

## Project Structure

- Login Page (User authentication)  
- Home Page (Main navigation)  
- Category Page (Item categories)  
- Product Page (Item details and actions)  
- Cart Page  (Dummy page until further change)
- Account Details Page  (Dummy page until further change)
- Help Page  (Dummy page until further change)

## How It Works

- User data is stored in a text file (`USERS.txt`)  
- Issues reported by users are stored in `issues.txt`  
- Application navigates between different pages using multiple frames
